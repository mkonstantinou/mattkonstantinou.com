Projects

Mini Dwarf Fortress Fall 2013

Dwarf Fortress Mini was a group project as part of Introduction to Game Development with Professor Andrew Grapsas. Working with four classmates, our goal was to recreate the classic game Dwarf Fortress as a text-based simulation in C++ from the ground up, including most (if not all) data structures. My contributions to the project include a JSON parsing library, state machine, actor factory, user input manager, and pathfinding system.

Reversi Fall 2014

Reversi was a group project as part of Artificial Intelligence class with Professor Noel Codella. Working with two other classmates, our goal was to create an algorithm that could win a game of Reversi in the fewest moves possible. Our implementation was able to solve the game in an average of 11 moves. My contributions to the project include the GUI, general game mechanics, and heuristic function.
